## Project Overview
This project analyzes student placement data using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Analysis

## Key Insights
- Overall placement rate: 54.46%
- Internship experience increases salary
- Higher CGPA improves placement probability
- Coding skills help Tier 3 students get better salaries

## Dashboard Features
- Placement Rate KPI
- Branch-wise placement chart
- Internship vs Salary analysis
- Backlog distribution
